# Event Hub Features

## Event Management
- Create, edit, and delete events
- Event calendar with date picker
- Event listing with pagination
- Event details page with full information
- Event status tracking (upcoming, ongoing, completed, cancelled)

## Categories & Filtering
- 8 event categories: BMW, JDM, Classics, Drift, Tuning, Racing, Meetup, Track Day
- Multi-category filtering
- Category-based event discovery
- Category icons and color coding

## Location & Maps
- Event location display
- Map integration with event markers
- Latitude/longitude coordinates
- Location-based event filtering

## Photos & Media
- Photo upload for events
- Photo gallery for each event
- Photo captions
- Photo likes and comments
- Drag-and-drop upload

## Comments & Ratings
- Event comments and reviews
- 5-star rating system
- Comment editing and deletion
- User avatars and timestamps
- Comment threading

## Attendee Management
- Event attendance tracking
- Attendee status (interested, attending, cancelled)
- Max attendees limit
- Current attendees count
- Attendee list

## Notifications
- Event reminder notifications
- Event update notifications
- New comment notifications
- New photo notifications
- Event cancellation notifications
- Notification settings
- Email notifications

## User Features
- User authentication (OAuth)
- User profiles
- User preferences
- Favorite categories
- Event history
- User dashboard

## UI/UX
- Modern, light design
- Responsive layout (mobile, tablet, desktop)
- Dark/light theme support
- Loading states
- Error handling
- Empty states
- Smooth animations
- Accessibility features
