# Usage Guide

## Creating an Event
1. Click "Create Event" button
2. Fill in event details
3. Select category and location
4. Set date and time
5. Submit form

## Finding Events
1. Browse calendar or event list
2. Use category filters
3. Search by location
4. Check map view

## Attending Events
1. Click on event
2. Click "Attend" button
3. Choose attendance status
4. Confirm

## Uploading Photos
1. Go to event details
2. Click "Upload Photo"
3. Select image file
4. Add caption (optional)
5. Submit

## Leaving Comments
1. Scroll to comments section
2. Write your comment
3. Rate event (optional)
4. Click "Post"

## Managing Notifications
1. Click bell icon
2. View all notifications
3. Mark as read
4. Delete if needed
