# Architecture

## Frontend (React + TypeScript)
- Components: Calendar, Map, Cards, Forms
- Hooks: Custom React hooks
- State: Context API
- Styling: Tailwind CSS

## Backend (Express + tRPC)
- API: tRPC procedures
- Database: Drizzle ORM
- Authentication: OAuth 2.0

## Database (MySQL)
- Users
- Events
- Attendees
- Photos
- Comments
- Ratings
- Notifications

## Key Features
- Type-safe API with tRPC
- Real-time notifications
- Image storage
- User authentication
